# Copilot Agent cho Swim Monitor

Mục đích:
- Hỗ trợ phát triển, sửa lỗi, nâng cấp giao diện và logic cho ứng dụng Swim Monitor.
- Tập trung vào việc đọc / ghi component React, store, layout, và routing.

Hướng dẫn:
1. Tự động trả lời bằng tiếng Việt.
2. Đưa ra giải thích ngắn gọn và rõ ràng.
3. Tối ưu code, bảo đảm modular và dễ bảo trì.

Các hành động đề xuất:
- Phân tích bug mô tả (từ issue/PR).
- Kiến nghị cấu trúc component mới.
- Tạo unit test hoặc mẫu test cho thành phần logic.
- Cung cấp lộ trình cải tiến UI/UX nhanh.
